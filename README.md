# Random Quote Generator

## Автор
Чиркин Виктор

## Описание
Приложение для генерации случайных цитат с возможностью фильтрации по автору и теме, сохранения истории в JSON и загрузки её обратно.

## Примеры использования
- Генерация 10 цитат подряд — все должны быть разными (или возможны повторы).
- Фильтрация по автору «Джон Леннон» — отображаются только его цитаты.
- Сохранение и загрузка истории — после перезапуска приложения история сохраняется.
